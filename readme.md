# Chess in C++

This project simulates a game of Chess in the terminal of your computer.

## Instructions for Build and Use

Steps to build and/or run the software:

1. It should just run if you open the folder and double-click "chess.exe"
2. Don't mess with the save.txt file, it will probably break the save/load feature if you do

Instructions for using the software:

1. To move a piece, type the start square and the end square of where you want that piece to move to. 
2. For example, to move a pawn from e2 to e4, you can type "e2e4", "e2, e4", "e2->e4" or anything like that.
3. Once in the game, type "help" for commands, "save" to save the game and "exit" to close the terminal.
